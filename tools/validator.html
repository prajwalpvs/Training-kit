Found Issues

! Missing alt attribute on image

All images must have an alternative text description to be accessible to screen reader users. Missing alt attributes violate WCAG 2.1 Success Criterion 1.1.1 (Level A).

<img src="company-logo.png">

How to Fix

Add a descriptive alt attribute to the image that conveys the same information:

<img src="company-logo.png" alt="Company Name Logo">
  • If the image is decorative, use an empty alt attribute: alt=""
  • Do not include "image of" or "picture of" in the alt text
  • Keep alt text concise but descriptive
! Generic link text

Links should have descriptive text that makes sense out of context. Generic link text like "click here" or "read more" does not provide sufficient context for screen reader users who navigate by links.

<a href="services.html">Click here</a> to view our services.

How to Fix

Rewrite the link to use descriptive text:

View our <a href="services.html">complete list of services</a>.
  • Avoid phrases like "click here," "read more," "learn more," etc.
  • Make the link text clearly indicate where the link will take the user
  • If the same link text is used multiple times, ensure each link goes to the same place
i Consider adding more headings

Long blocks of text without headings can be difficult to navigate, especially for users with cognitive disabilities or those using screen readers. Adding appropriate heading structure makes content more scannable and accessible.

Large paragraph block (253 words) without headings

Best Practice

Break up content with descriptive headings:

  • Use headings to separate different topics or sections
  • Maintain a proper heading hierarchy (h1 → h2 → h3)
  • Aim for paragraphs of 3-5 sentences for better readability
  • Consider using lists for related items or steps

Recommendations for Improvement

Based on the validation results, here are some recommendations to improve your content's accessibility:

Structure & Semantics

  • Use proper heading hierarchy (h1-h6) to organize content
  • Use semantic HTML elements like <nav>, <main>, etc.
  • Ensure proper list markup for all lists

Text Content

  • Improve reading level to around 8th grade
  • Break up long paragraphs into shorter chunks
  • Use clear and descriptive link text

Images & Media

  • Add descriptive alt text to all non-decorative images
  • Use empty alt text (alt="") for decorative images
  • Ensure all media has appropriate captions or transcripts

Forms & Controls

  • Associate all form controls with labels
  • Add clear error messages for form validation
  • Ensure all interactive elements are keyboard accessible
